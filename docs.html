What is Zenclora?

Zenclora is a Debian-based desktop Linux distribution optimized for stability and daily use. Zenclora offers simplified package and system management commands to make Linux usage easier; it avoids unnecessary complexity while providing control to the user.

Installation

1. Download the ISO file from the website. Copy it to your USB drive using Ventoy or Rufus.

2. Enter your motherboard's BIOS and boot your system from the USB drive.

Note: Zenclora supports most modern hardware. If your GPU is Nvidia, ensure it is compatible with nouveau.

Zenclora may not boot on very old hardware. A minimum of 4GB RAM is recommended. Additionally, problems may arise on very new hardware with low Linux compatibility.

Once you reach the Boot Screen, you can start in Live mode to try Zenclora, or start the Debian Installer to install it to your hard drive. Complete the necessary steps via the GUI, install your system, and start exploring!

Note for Live Mode: When you press shut down in Live mode, you may see the Plymouth screen. Simply press the Enter key; this will safely remove the live-medium and resolve the shutdown issue.

Zenclora Commands

No more getting lost searching for repos on the internet or risking your system with old or fake repos. Zenclora saves you from wasting hours with its special commands. You can download the software you want using up-to-date and official repos. At the same time, system management scripts will save you from getting lost in the terminal for hours!

System Management

To update the system quickly, easily, and safely:

sudo update

To easily change your DNS:

sudo change-dns

Clean up your system and free up space:

cleanup

Performance & Utility

Install the fast Liquorix Kernel:

install-liquorix

*Check compatibility with your Nvidia card.

Use Zram to increase performance if you have low RAM:

zram-settings

Run Local AI models:

install-ollama

Get detailed network information:

net-info

Get system information:

zenclora-fetch
